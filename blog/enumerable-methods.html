Cycle through your problems

September-19th-2015

Today, we will talk about Enumerable method. The enumerable method is a very important one in the programing language of Ruby. Enumerable is used on classes in ruby as a collection device. Every enumerable must be able to use the each method. The Each method runs through every item in your collection and acceses it for use. So lets say you have an array, or a place to store things. You have tickets to three different teams in your collection. Kings, Lakers, Clippers tickets. Now if you wanted to go through your collection and have it look at each ticket you would use the .each method. This would look like this in ruby

Tickets = ["Kings", "Lakers", "Clippers"]

Tickets.each do {|print| p print}

This would print each ticket for you, but going through each point in your Tickets collection, and printing them all out for you.

Now we need to look at one method called cycle. Cycle does what its name implys, it cycles through your code. It can take an argument which is an Integer that will tell it how many times to run your code. Or it can be left without an argument and it will run forever unless given a break command. The cycle code looks something like this

days_of_week = %w{Monday Tuesday Wednesday Thursday Friday Saturday Sunday} => ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"] # This will print the days of the week over and over, # forever until you stop it with control-c. days_of_week.cycle {|day| puts day} => Monday Tuesday Wednesday Thursday Friday Saturday Sunday Monday Tuesday ... # The cycle can be broken with the "break" statement days_of_week.cycle do |day| puts day break if day == "Friday" end => Monday Tuesday Wednesday Thursday Friday => nil

As you can see, it will go through your collection object and print it out as many times as you pass the argument to it.
